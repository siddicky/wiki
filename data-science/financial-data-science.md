# Financial

### Fundamentals

#### Average Return

$$R_{t,t+1} = {P_{t+1}-P_t \over P_t}$$

Average return is misleading because it disregards volatility.

#### Compounded Return

 $$R_{t,t+2} = (1+R_{t,t+1})(1+R_{t+1,t+2})-1$$ 

We can get the annualized returns by repeating this for every monthly return.
