# Electron

### Unpack ASAR

```text
npm install -g asar
asar extract core.asar <dst>
```

On some apps like MS Teams its already unpacked "C:\Users\xct\AppData\Local\Microsoft\Teams\current\resources\app.asar.unpacked\node\_modules"
